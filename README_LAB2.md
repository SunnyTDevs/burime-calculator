## 📸 Как сделать скриншот для отчета

### В IntelliJ IDEA:
1. Запустите все тесты (кнопка ▶ возле класса)
2. Дождитесь завершения (все тесты должны быть зелеными ✅)

## 🐛 Решение проблем

### Проблема: "No tests found"
**Решение:**
1. File → Invalidate Caches → Invalidate and Restart
2. После перезапуска: Gradle → Reload All Gradle Projects

### Проблема: SSL certificate error при запуске через Gradle
**Решение:**
Используйте запуск через IntelliJ IDEA (зеленая кнопка ▶), а не через командную строку.

### Проблема: Тесты не запускаются
**Решение:**
1. File → Project Structure → Project → SDK: выберите Java 17
2. Build → Rebuild Project
3. Gradle → Reload All Gradle Projects
