## 2. РЕАЛИЗОВАННЫЙ ФУНКЦИОНАЛ

### 2.1. Основной класс: `BubbleSort.java`

Расположение: `src/main/java/com/burime/calculator/BubbleSort.java`

**Реализованные методы:**

1. **`public static int[] sort(int[] array)`**
   - Назначение: Сортировка массива целых чисел по возрастанию
   - Алгоритм: Пузырьковая сортировка с оптимизацией (досрочный выход)
   - Особенности: Не изменяет исходный массив (создает копию)
   - Обработка ошибок: Выбрасывает `IllegalArgumentException` при `array == null`

2. **`public static int[] sortDescending(int[] array)`**
   - Назначение: Сортировка массива целых чисел по убыванию
   - Алгоритм: Пузырьковая сортировка с инверсией условия сравнения
   - Особенности: Не изменяет исходный массив (создает копию)
   - Обработка ошибок: Выбрасывает `IllegalArgumentException` при `array == null`

## 8. СЛОЖНОСТЬ АЛГОРИТМА

### Временная сложность:
- **Лучший случай:** O(n) - массив уже отсортирован (благодаря оптимизации)
- **Средний случай:** O(n²)
- **Худший случай:** O(n²) - массив отсортирован в обратном порядке

### Пространственная сложность:
- **O(n)** - из-за создания копии входного массива
