# Буриме-Калькулятор

Учебный проект для совместной разработки простого калькулятора на Java 17.

## Правила игры

1. Каждый участник добавляет **ровно одну строчку кода** за раз
2. Перед добавлением своей строки необходимо получить последние изменения (`git pull`)
3. После добавления строки - зафиксировать и отправить изменения (`git commit` и `git push`)

## Компиляция и запуск

### Через IntelliJ IDEA (рекомендуется)

1. Откройте проект в IntelliJ IDEA
2. IDE автоматически распознает Gradle проект и настроит его
3. Нажмите зеленую кнопку ▶️ запуска рядом с методом `main` в `Calculator.java`

### Через Gradle (командная строка)

```bash
# Запуск приложения
./gradlew run

# Или на Windows
gradlew.bat run
```

### Через javac (альтернативный способ)

```bash
# Компиляция
javac -d out src/main/java/com/burime/calculator/Calculator.java

# Запуск
java -cp out com.burime.calculator.Calculator
```

## Задание

Дописать логику калькулятора, который:
- Принимает два числа и операцию (+, -, *, /)
- Выполняет вычисление
- Выводит результат

## Проверка авторства

После завершения работы проверить авторство каждой строки:

```bash
git blame -u src/main/java/com/burime/calculator/Calculator.java
```

## Настройка имени пользователя

Каждый участник должен настроить свое имя для репозитория:

```bash
git config user.name "Ваше Имя"
git config user.email "your.email@example.com"
```
