### 3.1 Численное интегрирование

#### 3.1.1 Обобщённая формула трапеций

Формула трапеций для вычисления интеграла:

$$I_n = \frac{h}{2} \left[ f(x_0) + 2 \sum_{i=1}^{n-1} f(x_i) + f(x_n) \right]$$

где h = (b - a) / n — шаг разбиения, n — количество разбиений.

#### 3.1.2 Критерий Рунге

Для адаптивного выбора числа разбиений используется правило Рунге:

$$\Delta_{2n} = \frac{|I_{2n} - I_n|}{3}$$

Вычисления прекращаются, когда Δ₂ₙ ≤ ε.

#### 3.1.3 Реализация

```java
// Функциональный интерфейс для передачи функции f(x)
@FunctionalInterface
public interface DoubleUnaryOperator {
    double applyAsDouble(double value);
}

// Класс для интегрирования
public class Integrator {

    private static final double EPS = 0.0001;

    // Метод интегрирования с передачей функции через параметр
    public static double integrate(DoubleUnaryOperator f, double a, double b, double eps) {
        int n = 2;
        double i_n = trapezoid(f, a, b, n);
        double i_2n = trapezoid(f, a, b, n * 2);

        while (Math.abs(i_2n - i_n) / 3 > eps) {
            n *= 2;
            i_n = i_2n;
            i_2n = trapezoid(f, a, b, n * 2);
        }

        return i_2n;
    }

    // Обобщённая формула трапеций
    private static double trapezoid(DoubleUnaryOperator f, double a, double b, int n) {
        double h = (b - a) / n;
        double sum = f.applyAsDouble(a) + f.applyAsDouble(b);

        for (int i = 1; i < n; i++) {
            sum += 2 * f.applyAsDouble(a + i * h);
        }

        return sum * h / 2;
    }
}
```

### 3.2 Обработка списков (MAP и FOLD)

#### 3.2.1 MAP (отображение)

Функция MAP применяет преобразующую функцию к каждому элементу коллекции:

```
MAP: [x₁, x₂, x₃, ..., xₙ] → [f(x₁), f(x₂), f(x₃), ..., f(xₙ)]
```

**Реализация:**

```java
public static double[] map(double[] array, DoubleUnaryOperator func) {
    double[] result = new double[array.length];
    for (int i = 0; i < array.length; i++) {
        result[i] = func.applyAsDouble(array[i]);
    }
    return result;
}
```

**Результаты применения MAP:**

| Функция | Вход | Выход |
|---------|------|-------|
| F(x) = eˣ | [1, 2, 3, 4, 5] | [2.7183, 7.3891, 20.0855, 54.5982, 148.4132] |
| F(x) = sin(x) | [1, 2, 3, 4, 5] | [0.8415, 0.9093, 0.1411, -0.7568, -0.9589] |
| F(x) = floor(x) | [1.7, 2.3, 3.8] | [1, 2, 3] |

#### 3.2.2 FOLD (свёртка)

Функция FOLD свёртывает коллекцию в одно значение с помощью бинарной функции:

```
FOLD: [x₁, x₂, x₃, ..., xₙ] → f(f(f(x₁, x₂), x₃), ..., xₙ)
```

**Реализация:**

```java
public static double fold(double[] array, DoubleBinaryOperator func, double identity) {
    double result = identity;
    for (double value : array) {
        result = func.applyAsDouble(result, value);
    }
    return result;
}
```

**Результаты применения FOLD:**

| Функция | Вход | Identity | Результат |
|---------|------|----------|-----------|
| F(x,y) = x·y | [1, 2, 3, 4, 5] | 1.0 | 120.0 (5! = 1·2·3·4·5) |
| F(x,y) = min(x,y) | [5, 3, 8, 1, 9] | MAX_VALUE | 1.0 |
| F(x,y) = √(x·y) | [1, 4, 9, 16] | 1.0 | 24.0 |

#### 3.2.3 Комбинирование MAP и FOLD

MAP и FOLD можно комбинировать для решения сложных задач:

```java
// Вычислить произведение e^(x) для x ∈ [1, 2, 3]
double[] data = {1.0, 2.0, 3.0};
double[] expData = map(data, Math::exp);                    // [e, e², e³]
double product = fold(expData, (x, y) -> x * y, 1.0);      // e · e² · e³ = e⁶
```

## 4. Ответы на контрольные вопросы

### 1. Что такое делегат?

**Делегат (delegate)** в C# — это тип данных, который инкапсулирует ссылку на метод с определённой сигнатурой. В Java аналогом являются **функциональные интерфейсы** — интерфейсы с единственным абстрактным методом, которые могут быть реализованы с помощью лямбда-выражений.

### 2. Для чего предназначен делегат?

Делегаты предназначены для:
- Передачи методов как параметров в другие методы (функции высшего порядка)
- Реализации механизма обратных вызовов (callbacks)
- Создания событийной модели
- Гибкой настройки поведения алгоритмов без наследования

### 3. Как создать экземпляр делегата?

В Java создание экземпляра функционального интерфейса:

```java
// С помощью лямбда-выражения
DoubleUnaryOperator f = x -> x * x;

// С помощью ссылки на метод
DoubleUnaryOperator f = Math::sqrt;

// Анонимный класс
DoubleUnaryOperator f = new DoubleUnaryOperator() {
    @Override
    public double applyAsDouble(double x) {
        return x * x;
    }
};
```

### 4. Какие операции над делегатами вы знаете?

Основные операции:
- **Комбинирование** (Combine): объединение нескольких делегатов в цепочку
- **Удаление** (Remove): `-` оператор для удаления метода из цепочки
- **Мультикаст** (Multicast): вызов нескольких методов одновременно

### 5. Что такое событие?

**Событие (event)** — это механизм оповещения, позволяющий классу уведомлять другие объекты о произошедших изменениях. События основаны на механизме делегатов и реализуют паттерн **Observer (Наблюдатель)**.

### 6. Для чего предназначены события?

События используются для:
- Уведомления об изменении состояния объекта
- Реализации слабой связанности между компонентами системы
- Обработки пользовательских действий (клики, нажатия клавиш)
- Асинхронного оповещения

### 7. Какие способы определения событий вы знаете?

В C# события могут быть определены:
- С использованием встроенного делегата `EventHandler`
- С использованием пользовательского делегата
- С обобщённым `EventHandler<T>` для типизированных аргументов

В Java используется механизм **Listener/Observer** с функциональными интерфейсами.

### 8. Что такое функция высшего порядка?

**Функция высшего порядка (Higher-Order Function)** — это функция, которая:
- Принимает одну или несколько функций как аргументы, и/или
- Возвращает функцию как результат

Примеры: `map()`, `filter()`, `fold()`.

### 9. Как работает функция MAP?

MAP применяет преобразующую функцию к каждому элементу коллекции:

```
┌──────────────────────────────────────────┐
│  Исходная коллекция: [x₁, x₂, x₃, x₄]    │
│                    ↓                     │
│  Функция f применяется к каждому xᵢ      │
│                    ↓                     │
│  Результат: [f(x₁), f(x₂), f(x₃), f(x₄)]  │
└──────────────────────────────────────────┘
```

### 10. Как работает функция FOLD?

FOLD свёртывает коллекцию в одно значение, последовательно применяя бинарную функцию:

```
┌──────────────────────────────────────────────────────────┐
│  Исходная коллекция: [x₁, x₂, x₃, x₄]                    │
│                                                              │
│  Шаг 1: result = f(x₁, x₂)  →  промежуточное значение      │
│  Шаг 2: result = f(result, x₃)  →  новое значение         │
│  Шаг 3: result = f(result, x₄)  →  финальное значение      │
│                                                              │
│  Финальный результат: одно значение                         │
└──────────────────────────────────────────────────────────┘
```

### 11. Какие виды функций FOLD вы знаете?

1. **foldl (foldLeft)** — свёртка слева направо
   ```
   f(f(f(f(x₁, x₂), x₃), x₄)
   ```

2. **foldr (foldRight)** — свёртка справа налево
   ```
   f(x₁, f(x₂, f(x₃, x₄)))
   ```

3. **fold с identity** — свёртка с начальным значением
   ```
   f(f(f(f(identity, x₁), x₂), x₃)
   ```

Важно: для некоммутативных операций `foldl` и `foldr` дают разные результаты.

## Приложение: Листинги программ

### Integrator.java

```java
package com.burime.calculator;

import java.util.function.DoubleUnaryOperator;

public class Integrator {

    private static final double DEFAULT_EPS = 0.0001;

    public static double integrate(DoubleUnaryOperator f, double a, double b) {
        return integrate(f, a, b, DEFAULT_EPS);
    }

    public static double integrate(DoubleUnaryOperator f, double a, double b, double eps) {
        IntegrationResult result = integrateWithStats(f, a, b, eps);
        return result.value;
    }

    public static IntegrationResult integrateWithStats(DoubleUnaryOperator f, double a, double b, double eps) {
        int n = 2;
        double i_n = generalizedTrapezoid(f, a, b, n);
        double i_2n = generalizedTrapezoid(f, a, b, n * 2);
        int iterations = 1;

        while (Math.abs(i_2n - i_n) / 3 > eps) {
            n *= 2;
            i_n = i_2n;
            i_2n = generalizedTrapezoid(f, a, b, n * 2);
            iterations++;
        }

        double rungeError = Math.abs(i_2n - i_n) / 3;
        return new IntegrationResult(i_2n, iterations, n * 2, rungeError);
    }

    public static double generalizedTrapezoid(DoubleUnaryOperator f, double a, double b, int n) {
        double h = (b - a) / n;
        double sum = 0.0;

        for (int i = 0; i <= n; i++) {
            double x_i = a + i * h;
            double coefficient = (i == 0 || i == n) ? 1.0 : 2.0;
            sum += coefficient * f.applyAsDouble(x_i);
        }

        return (h / 2.0) * sum;
    }
}
```

### ListProcessor.java

```java
package com.burime.calculator;

import java.util.function.DoubleBinaryOperator;
import java.util.function.DoubleUnaryOperator;
import java.util.ArrayList;
import java.util.List;
import java.util.Arrays;

public class ListProcessor {

    // MAP - отображение элементов
    public static double[] map(double[] array, DoubleUnaryOperator func) {
        if (array == null) {
            throw new IllegalArgumentException("Массив не может быть null");
        }
        double[] result = new double[array.length];
        for (int i = 0; i < array.length; i++) {
            result[i] = func.applyAsDouble(array[i]);
        }
        return result;
    }

    // FOLD - свёртка элементов
    public static double fold(double[] array, DoubleBinaryOperator func, double identity) {
        if (array == null) {
            throw new IllegalArgumentException("Массив не может быть null");
        }
        double result = identity;
        for (double value : array) {
            result = func.applyAsDouble(result, value);
        }
        return result;
    }
}
```
